# Zero — 从零开始你的一人公司

AI 工作流引擎。输入一个搞钱目标，自动跑完画像分析、市场扫描、匹配打分、行动计划，输出一份完整的商业分析报告。

---

## 是什么

**Zero 不是聊天机器人。** 它是工作流引擎。

| | 聊天机器人 | Zero |
|---|---|---|
| 用户做什么 | 一轮一轮聊 | 一次输入 + 填完信息 |
| AI做什么 | 回一句等下一句 | 连续跑 5 个节点不等任何人 |
| 产出 | 聊天记录 | 一份结构化报告 |
| 能卖钱吗 | 很难 | 有人愿意为报告付钱 |

---

## 怎么运行

### 方式一：直接打开（最简单）

双击 `index.html`，浏览器里直接运行。

### 方式二：本地服务器（推荐）

```bash
# 用 Python 起一个本地服务器
cd zero
python -m http.server 3000

# 或用 Node.js
npx serve zero

# 然后打开 http://localhost:3000
```

### 方式三：部署到公网（免费）

1. 把整个 zero 文件夹上传到 [Vercel](https://vercel.com) 或 [Netlify](https://netlify.com)
2. 或传到 GitHub → 开启 GitHub Pages
3. 获得一个公开链接，任何人都能访问

---

## 需要准备什么

### DeepSeek API Key（必须）

1. 打开 [platform.deepseek.com](https://platform.deepseek.com)
2. 注册账号 → API Keys → 创建新的 API Key
3. 充值 ¥10（够用几千次分析）
4. 在 Zero 设置里填入 API Key

**费用：** 每次完整分析约 ¥0.01-0.03（取决于报告详细程度）

## 两个模式

### 🧭 不知道方向
回答 5 个问题 → 工作流自动分析 → 输出 TOP3 推荐方向 + 30天行动计划

### 🎯 已有想法
输入你的想法 → 工作流评估可行性 → 识别风险 → 生成执行计划

## 文件结构

```
zero/
├── index.html      # 主应用（包含全部前端代码）
├── manifest.json   # PWA 配置（可添加到手机桌面）
├── sw.js           # Service Worker（离线缓存）
└── README.md       # 本文件
```

---

## 技术栈

- 纯原生 HTML/CSS/JS（零依赖、零框架）
- DeepSeek API（工作流引擎）
- PWA（可安装到手机/电脑桌面）
- localStorage（本地存储历史报告）
